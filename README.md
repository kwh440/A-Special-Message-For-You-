# Romantic Digital Experience

A premium, emotionally captivating single-page website designed to ask someone out. Built with HTML, CSS, and GSAP for a cinematic feel.

## 🚀 Features
- **Luxury Aesthetic**: Dark mode with soft pink glows and glassmorphism.
- **Cinematic Hero**: Autoplay video background with elegant text reveals.
- **Interactive Storytelling**: Parallax scroll sections with high-quality media.
- **Micro-interactions**: A playful "No" button that moves away and a heart-filled celebration for "Yes".

## 🛠️ Tech Stack
- **Structure**: Semantic HTML5
- **Style**: Custom Vanilla CSS
- **Animations**: GSAP (GreenSock Animation Platform)
- **Icons/Emoji**: Native system support

## 📦 Deployment Instructions (GitHub Pages)

1. **Create a New Repository**: Create a new repository on GitHub (e.g., `special-question`).
2. **Upload Files**: Upload all files from the `github-deployment` folder (including the `assets` folder) to the root of your repository.
3. **Enable Pages**:
   - Go to **Settings** > **Pages**.
   - Under **Build and deployment**, set the source to **Deploy from a branch**.
   - Select the **main** branch and the **/(root)** folder.
   - Click **Save**.
4. **Visit Your Site**: In a few minutes, your site will be live at `https://<your-username>.github.io/<repository-name>/`.
